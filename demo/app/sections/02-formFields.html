Input

fmInput
- change values of an input field
Actions:
I type "This text is " on the field "#fmInput"
I append "appended on [fmInput]..." on the field "#fmInput"
I clear the field "#fmInput"
Checks:
I expect the field "#fmInput" value to not contain "[fmInput]"
I expect the field "#fmInput" value to be "This text is appended on [fmInput]..."
I expect the field "#fmInput" value to be empty
